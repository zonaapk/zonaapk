Descripción de Google Chrome

Google Chrome es un navegador web rápido, seguro y fácil de usar. Está diseñado para Android e incluye noticias personalizadas, enlaces rápidos a tus sitios web favoritos y descargas. Además, la Búsqueda y el Traductor de Google están integrados. Descárgalo ya para disfrutar de la misma experiencia con tu navegador web favorito en todos tus dispositivos. Navega rápido y escribe menos. Selecciona alguno de los resultados de búsqueda personalizados que se muestran automáticamente mientras escribes y accede rápidamente a las páginas web que ya has visitado. Utiliza la función Autocompletar para rellenar formularios de forma rápida. Navega de incógnito. Usa el modo de incógnito para navegar por Internet sin guardar el historial. Navega de forma privada en todos tus dispositivos. Accede a tu Chrome en diferentes dispositivos. Cuando inicias sesión en Chrome, puedes guardar tus marcadores, contraseñas y más en tu cuenta de Google para acceder a ellos en tus otros dispositivos. Solo necesitas un toque para acceder a tu contenido favorito. Chrome no solo funciona rápido con la Búsqueda de Google, sino que está diseñado para que disfrutes de tu contenido favorito con un solo toque. Accede a las redes sociales o tus sitios web de noticias favoritos directamente desde la página Nueva pestaña. La función "Tocar para buscar" está integrada en todas las páginas web de Chrome. Toca cualquier palabra o frase para iniciar una búsqueda de Google sin salir de la página en la que estás. Protege tu teléfono con Navegación Segura de Google. Navegación Segura de Google está integrada en Chrome. Protege tu teléfono con advertencias que se muestran si intentas acceder a sitios web peligrosos o descargar archivos dañinos. Descarga rápidamente y consulta páginas web y vídeos sin conexión. Chrome incluye un botón específico para que puedas descargar vídeos, imágenes y páginas web completas con un solo toque. También incluye una sección de descargas dentro del navegador, desde la que puedes acceder a todo el contenido descargado aunque no tengas conexión. Búsqueda por voz de Google. Chrome es un navegador web con el que puedes hablar. Usa tu voz para buscar respuestas estés donde estés sin tener que escribir ni utilizar las manos. Navega más rápido con tu voz donde y cuando quieras. Traductor de Google integrado: traduce páginas web completas de forma rápida. El Traductor de Google está integrado en Chrome para que puedas traducir sitios web completos a tu idioma con un solo toque. Recomendaciones personalizadas inteligentes. Chrome te ofrece una experiencia personalizada según tus intereses. En la página Nueva pestaña, encontrarás artículos que Chrome ha seleccionado basándose en tu historial de navegación.
